# 仕様

## UI仕様
- スマートフォンでPWAとして開いたとき、UIが画面いっぱいに最大化されること。枠は不要。
- 画面は上段・中段・下段の３段構成
    - 上段
        - Stack エリア。計算中のスタックの状態を表示。
        - stack のトップ（つまり最新の入力値）が一番下に来る並び順で数値のリストを表示。
        - 配置は、左寄せ、上寄せ。
    - 中段
        - Command エリア。ここに四則演算や関数などのボタンを並べる。
        - 1行目
            - "CLEAR"   ... stack をクリアする
            - ”DEL"     ... stack を pop する
        - 2行目
            - 四則演算ボタン (+, -, *, / を横に並べる)

        - テキストボックスの右側に Enter ボタン
    - 下段
        - Input エリア。テキストボックスとテンキーを配置。
        - 一番上に、入力中の数字が表示されるテキストボックスと、その右に "ENTER" ボタン。
        - その下にテンキーを配置。
            - 3列 x 4行 のよくある構成
            - 4行目は "0", ".", "BACK"
