# 仕様

## UI仕様
- スマートフォンでPWAとして開いたとき、UIが画面いっぱいに最大化されること。枠は不要。
- 画面は上段・中段・下段の３段構成
    - 上段
        - Stack エリア。計算中のスタックの状態を表示。
        - stack のトップ（つまり最新の入力値）が一番下に来る並び順で数値のリストを表示。
        - 配置は、左寄せ、上寄せ。
    - 中段
        - Input エリア。
        - 入力中の数字が表示されるテキストボックス
        - テキストボックスの右側に Enter ボタン
    - 下段
        - Keypad エリア。
        - 上から次の順序でボタンを配置。
            1. テンキー（3列 x 4行 のよくある構成で、4行目は "0", ".", "BACK"）
            2. 四則演算ボタン (+, -, *, / を横に並べる)
            3. その他の関数（これは今後追加。今はなくて良い）
